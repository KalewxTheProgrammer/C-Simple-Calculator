# C++ Simple Calculator

A beginner-friendly command-line calculator built in C++.  
Supports addition, subtraction, multiplication, and division.

---

## About the Author
I'm a 15-year-old self-taught programmer passionate about coding.
This is one of my first C++ projects — more coming soon!

---

## Features
- Add, subtract, multiply, and divide two numbers
- Handles division by zero with an error message
- Clean and simple terminal interface

---

## How to Compile and Run

### Requirements
- A C++ compiler (g++)
- Windows, Mac, or Linux

### Windows (PowerShell)
```bash
g++ calculator.cpp -o calculator
./calculator
```

### Mac / Linux (Terminal)
```bash
g++ calculator.cpp -o calculator
./calculator
```

## How to Use

1. Run the program
2. Enter the first number when prompted
3. Enter the second number when prompted
4. Enter one of the following operators:

| Operator | Operation      |
|----------|---------------|
| `+`      | Addition       |
| `-`      | Subtraction    |
| `*`      | Multiplication |
| `/`      | Division       |
